<h2 id="introduction">Introductie</h2>

Voor het weergeven van notificaties binnen tabellen.

<h2 id="quickstart">Snelstart</h2>

SCSS importeren:

```css
@use "@minvws/manon/notification";
@use "@minvws/manon/notification-table";
```
