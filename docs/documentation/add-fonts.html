Lettertype toevoegen

Overzicht van de stappen om een eigen font toe te voegen.

  1. Voeg de gewenste lettertypes toe aan het project. Bijvoorbeeld binnen een fonts-map.
  2. Kopieer het voorbeeldbestand manon/scss/fonts en plaats het op de gewenste plek binnen het project.
  3. Vul het voorbeeldbestand in met de gekozen lettertypes. Voeg bij src: het pad toe naar de toegevoegde lettertypes binnen het project. Voor meer informatie zie: Voorbeelden lettertypes toevoegen.

Voorbeelden

Lettertype

              
@font-face {
  font-family: "Naam van het font";
  font-weight: normal;
  font-style: normal;
  src: url("../fonts/fontnaam.woff2") format("woff2"),
    url("../fonts/fontnaam.woff") format("woff");
}

Lettertype met verschillende varianten

              
/* Naam van het font */
@font-face {
  font-family: "Naam van het font";
  font-weight: normal;
  font-style: normal;
  src: url("../fonts/fontnaam.woff2") format("woff2"),
    url("../fonts/fontnaam.woff") format("woff");
}

@font-face {
  font-family: "Naam van het font";
  font-weight: normal;
  font-style: italic;
  src: url("../fonts/fontnaam-italic.woff2") format("woff2"),
    url("../fonts/fontnaam-italic.woff") format("woff");
}

Icoonlettertype

              
/* Manon icon font */
@font-face {
  font-family: "Manon icons";
  font-weight: normal;
  font-style: normal;
  src: url("../fonts/manon-icons-0.1.eot");
  src: url("../fonts/manon-icons-0.1.woff") format("woff"),
    url("../fonts/manon-icons-0.1.ttf") format("truetype"),
    url("../fonts/manon-icons-0.1.svg") format("svg");
}
