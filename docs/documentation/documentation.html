Manon installeren (lokaal docs bekijken)

Manon maakt gebruik pnpm, een package die beschikbaar is via npm.

  1. Manon downloaden: git clone https://github.com/minvws/nl-rdo-manon.git
  2. Open de folder waar Manon is ge√Ønstalleerd, bijvoorbeeld: cd nl-rdo-manon
  3. Bekijk de pnpm installatie instructies
  4. Dependencies installeren: pnpm install
  5. Open de Manon documentatie folder: cd docs
  6. SCSS compilen naar CSS: pnpm run build
  7. De documentatie is nu lokaal te bekijken door het bestand nl-rdo-manon/docs/index.html te openen in je webbrowser
