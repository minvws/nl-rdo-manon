Zijmenu

Bij lange documenten of pagina's kan een zijmenu gebruikt worden om het navigeren voor de gebruiker toegankelijk en overzichtelijk te houden.

Voorbeelden

Zijmenu

Benodigde bestanden

  • manon/scss/sidemenu-base
  • manon/scss/sidemenu-base-variables

Benodigde stappen

  1. Voeg de class sidemenu toe aan de main.
  2. Voeg direct binnen de main een nav toe met de gewenste inhoud.
  3. Groepeer de overige content in een enkel element indien dat nog niet gebeurd is. Bijvoorbeeld: div, article of section. Voor meer informatie zie: Voorbeeld sidemenu structuur.

Aandachtspunten

  • Deze versie is een zijmenu zonder inklapfunctionaliteit. Deze versie maakt geen gebruik van JavaScript. Als er wel gekozen is voor een inklapbaar menu wordt deze variant ook getoont als een gebruiker geen JavaScript heeft.

Inklapbaar zijmenu

Het menu is standaard uitgeklapt en kan door de gebruiker ingeklapt worden.

Benodigde bestanden

  • manon/scss/sidemenu-base
  • manon/scss/sidemenu-base-variables
  • manon/js/sidemenu

Benodigde stappen

  1. Voeg de class sidemenu toe aan de main.
  2. Voeg direct binnen de main een nav toe met de gewenste inhoud.
  3. Groepeer de overige content in een enkel element indien dat nog niet gebeurd is. Bijvoorbeeld: div, article of section. Voor meer informatie zie: Voorbeeld sidemenu structuur.
  4. Voeg de referentie naar het Javascript bestand, sidemenu.js, toe aan de HTML-pagina's die gebruik dienen te maken van het inklapbare zijmenu. Voor meer informatie zie: JavaScript referentie toevoegen.

Aandachtspunten

  • De knop wordt automatisch toegevoegd. Deze hoeft dus niet handmatig in de html opgenomen te worden.
  • Zonder JavaScript wordt de knop niet toegevoegd en wordt het zijmenu getoont zonder deze in te klappen.
  • Indien er gekozen wordt voor een sluit- en open-knop als icoonknop zonder tekst. Voeg dan altijd de knoptekst in de taal van het document toe voor screenreader gebruikers.
    1. Voeg de knoptekst toe binnen een data-open-label. Bijvoorbeeld: data-open-label="Zijbalknavigatie".
    2. Voeg de knoptekst toe binnen een data-close-label. Bijvoorbeeld: data-open-label="Sluit zijbalknavigatie".
    3. Voeg een kopie van het variabelenbestand manon/scss/sidemenu/sidemenu-base-variables toe aan het project. Voor meer informatie zie: Componenten gebruiken en styling aanpassen - De styling van een een component aanpassen.
    4. Vul de variabele --sidemenu-base-open-button-icon in.
    5. Vul de variabele --sidemenu-base-close-button-icon in.
    6. Zet de variabele --sidemenu-base-button-font-size op 0 om de tekst visueel te verbergen.

Uitklapbaar zijmenu

Het menu is standaard ingeklapt en kan door de gebruiker uitgeklapt worden.

Benodigde bestanden

  • manon/scss/sidemenu-base
  • manon/scss/sidemenu-base-variables
  • manon/js/sidemenu

Benodigde stappen

  1. Voeg de class sidemenu toe aan de main.
  2. Voeg hiervoor de class sidemenu-closed toe aan de <main>.
  3. Voeg direct binnen de main een nav toe met de gewenste inhoud.
  4. Groepeer de overige content in een enkel element indien dat nog niet gebeurd is. Bijvoorbeeld: div, article of section. Voor meer informatie zie: Voorbeeld sidemenu structuur.
  5. Voeg de referentie naar het Javascript bestand, sidemenu.js, toe aan de HTML-pagina's die gebruik dienen te maken van het inklapbare zijmenu. Voor meer informatie zie: JavaScript referentie toevoegen.

Aandachtspunten

  • De knop wordt automatisch toegevoegd. Deze hoeft dus niet handmatig in de html opgenomen te worden.
  • Zonder JavaScript wordt de knop niet toegevoegd en wordt het zijmenu getoont zonder deze in te klappen.
  • Indien er gekozen wordt voor een sluit- en open-knop als icoonknop zonder tekst. Voeg dan altijd de knoptekst in de taal van het document toe voor screenreader gebruikers.
    1. Voeg de knoptekst toe binnen een data-open-label. Bijvoorbeeld: data-open-label="Zijbalknavigatie".
    2. Voeg de knoptekst toe binnen een data-close-label. Bijvoorbeeld: data-open-label="Sluit zijbalknavigatie".
    3. Voeg een kopie van het variabelenbestand manon/scss/sidemenu/sidemenu-base-variables toe aan het project. Voor meer informatie zie: Componenten gebruiken en styling aanpassen - De styling van een een component aanpassen.
    4. Vul de variabele --sidemenu-base-open-button-icon in.
    5. Vul de variabele --sidemenu-base-close-button-icon in.
    6. Zet de variabele --sidemenu-base-button-font-size op 0 om de tekst visueel te verbergen.

Aandachtspunten

  • Indien het JavaScript niet geladen kan worden is het zijmenu standaard uitgeklapt en niet in- of uit-klapbaar.
