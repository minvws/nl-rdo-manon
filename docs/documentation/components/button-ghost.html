Ghost button

Subtiele knopweergave.

  1. Voeg de class ghost toe om gebruik te maken van de visuele weergave van een "ghost button".

button

Visuele weergave:

Html-voorbeeld:

                
<button class="ghost">Lorem ipsum</button>

button uitgeschakeld

Visuele weergave:

Html-voorbeeld:

                
<button class="ghost" disabled>button primary</button>

input type="button"

Visuele weergave:

Html-voorbeeld:

                
<input type="button" value="Lorem ipsum" class="ghost">

input type="button" uitgeschakeld

Visuele weergave:

Html-voorbeeld:

                
<input type="button" value="Lorem ipsum" class="ghost" disabled>
