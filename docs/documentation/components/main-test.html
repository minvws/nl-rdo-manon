Content in een content-wrapper binnen main

De content wrapper zorgt ervoor dat de achtergrond door kan lopen terwijl de content een maximale breedte heeft.

Voor visueel voorbeeld zie: Content wrapper div

HTML-voorbeeld:

              
<main>
  <div> <!-- This can be full width div, section or article -->
    <div> <!-- While the content width can be limited -->
      <!-- content -->
    </div>
  </div>
</main>
