Icoonset toevoegen

  1. Maak een icoonlettertype of importeer een icoonlettertype. Voor meer informatie zie: Iconenset aanmaken
  2. Bij een eigen icoonlettertype of bij het importeren van een iconenset zonder gedefinieerde classes voeg een referentielijst toe. Voor meer informatie zie: referentielijst opstellen.
  3. Voeg bij een eigen opgestelde referentielijst het bestand toe aan het project via manon.scss. Voor meer informatie zie: Styling importeren.
  4. Voeg het icoonlettertype toe aan het project binnen: fonts.scss. Kopieer het voorbeeldbestand manon/scss/fonts als het project nog geen fonts.scss-bestand heeft.
  5. Gebruik de iconen uit de set. Voor meer informatie zie: Iconen gebruiken

Referentielijst aanmaken

Gebruik het voorbeelddocument: manon/scss/icon/icon-set-example.scss.

  1. Kopieer het voorbeeldbestand en geef het nieuwe bestand een unieke bestandsnaam.
  2. Laad het nieuwe referentiebestand in binnen het project via manon.scss
  3. Maak binnen de :root de css-variable aan met de referentie naar het bijbehorende unicode binnen het lettertype. Unicodes kunnen vaak opgevraagd worden binnen de tool waarmee het lettertype gemaakt wordt. Of door het lettertype in te laden binnen lettertype-bewerktsoftware. Let op: de variable dient de unicode inclusief dubbele aanhalingstekens te bevatten. Voorbeeld: "\e900".
  4. Voeg de referentie toe binnen het pseudo-element. In het voorbeeld wordt er gebruik gemaakt van een :before een :after of beide is ook mogelijk. Voeg indien gewenst ook een :after toe. Voor meer informatie zie: pseudo-elementen toevoegen
  5. Voeg binnen het content-element de eerder aangemaakte variabele toe. Bijvoorbeeld: var(--icon-cat).

Ter info:

  • @extend %icon-text-styling voegt de styling toe zoals deze binnen @minvws/manon/icon-base gedefinieerd staat.

Voorbeelden:

CSS voorbeeld:

              
  :root {
    --icon-cat: "\e900";
  }

  .icon {
    &:before {
      @extend %icon-text-styling;
    }

    &.icon-cat:before {
      content: var(--icon-cat);
      font-family: "Manon icons";
    }
  }
