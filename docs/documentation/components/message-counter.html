Notificatie-teller

Gebruik om de gebruiker te attenderen en te informeren over nieuwe berichten of notificaties.

  • Voeg de class "message-counter" toe om het aantal berichten visueel weer te geven als een notificatie-teller.
  • Voeg aria-label="" toe met een duidelijke omschrijving toe voor gebruiker die gebruik maken van screenreaders. Bijvoorbeeld: "Nieuwe berichten", "Ongelezen berichten" of "Nieuwe berichten binnen afspraken". Houdt de omschrijving kort maar zorg voor voldoende context.
