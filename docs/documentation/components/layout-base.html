Basis-layout instellen

Binnen het bestand layout-base-variables.scss kunnen algemene stijlkeuzes ingevuld worden die door alle hoofdelementen gebruikt worden zodra dit variabelen-bestand aanwezig is. Ieder hoofdelement heeft een eigen basisbestand waarbinnen unieke waardes ingevuld kunnen worden voor het specifieke element. Voor meer informatie zie Basisweergave kiezen en instellen.

Voor meer informatie over importeren en instellen van componenten. Zie: Componenten gebruiken en styling toevoegen

Benodigd

  • layout-base-variables.scss

Optioneel

Voor het volledige overzicht van bestanden die op basis van dit variabelen-bestand werken zie: Basisweergaven.

Instelbare variabelen

  • Algemene weergave:
    • padding-top
    • padding-right
    • padding-bottom
    • padding-left
    • max-width
    • Op titels en subtitels:
      • margin-top
      • margin-bottom
  • Witruimte tussen elementen:
    • Content-blok:
      • gap
    • Binnen dieper gelegen blok-elementen:
      • gap
