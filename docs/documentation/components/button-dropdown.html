Inklapbaar menu

  1. Voeg de benodigde bestanden toe aan het project. Voor een overzicht van de benodigde en optionele bestanden zie: Bijbehorende bestanden.
  2. Voeg de benodigde HTML toe. Voor meer informatie zie: Voorbeelden.

button

Visuele weergave:

  • Item 1
  • Huidige selectie
  • Item 3

Html-voorbeeld:

              
<div 
  data-open-label="Dropdown menu" 
  data-close-label="Sluit dropdown menu" 
  data-media="(min-width: 42rem)" 
  aria-label="Acties"
  class="collapsible">

  <div class="collapsing-element">
      <ul>
          <li><a href="#">Item 1</a></li>
          <li><a href="#" aria-selected="true">Huidige selectie</a></li>
          <li><a href="#">Item 3</a></li>
      </ul>
  </div>
</div>

  <button aria-haspopup="listbox" aria-current="true" aria-expanded="false" aria-controls="collapsible-language-selector">
    EN <i class="icon icon-descending">Lorem ipsum</i>
  </button>
  <div id="collapsible-language-selector">
      <ul>
          <li><a href="#">Item 1</a></li>
          <li><a href="#" aria-selected="true">Huidige selectie</a></li>
          <li><a href="#">Item 3</a></li>
      </ul>
  </div>
</div>
