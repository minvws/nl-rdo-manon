Gegroepeerde content binnen tegels

Aandachtspunten:

  • Voeg class="tile" toe voor de Gegroepeerde content binnen tegels op een enkel element. Voor meer informatie zie: tegelweergave
  • Voeg de benodigde bestanden toe aan het project. Voor een overzicht van de benodigde en optionele bestanden zie: Bijbehorende bestanden.

Voorbeelden

Enkel element div

Visueel voorbeeld:

Lorem ipsum dolor set

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras accumsan non justo sit amet convallis. Phasellus auctor hendrerit fermentum. Ut et dui sapien.

Html-voorbeeld:

              
<div class="tile" role="group">
  <h1>Lorem ipsum dolor set</h1>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras accumsan non justo sit amet convallis. Phasellus auctor hendrerit fermentum. Ut et dui sapien.</p>
  <div>
    <button>Lorem</button>
    <button>Ipsum</button>
  </div>
</div>

Instelbare variabelen

  • flex-direction
  • gap
  • border-width
  • border-style
  • border-color
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
