Binnen HTML

Iconen als afbeelding toevoegen.

HTML-voorbeeld:

Via CSS

Het onderstaande voorbeeld bevat een span. Andere html-attributen zijn ook mogelijk. Onder andere: a, p, button.

Let op: Voeg eerst een iconenset toe aan het project met indien nodig een eigen referentielijst. Voor meer informatie zie: Iconset toevoegen

Visueel voorbeeld:

HTML-voorbeeld:

              
<button class="icon icon-cat">Kat</button>
